# Automated Publishing

## Problem
Publishing @conduction/nextcloud-vue to npm requires manual version bumps, builds, and npm publish commands, which is error-prone and inconsistent.

## Solution
Add semantic-release with GitHub Actions to automate versioning and npm publishing on every push to main.
