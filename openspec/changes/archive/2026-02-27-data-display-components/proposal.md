# Proposal: Data Display Components

## Problem
No card-based view alternative to CnDataTable, and no smart cell renderer for type-aware value formatting.

## Solution
Add CnCardGrid for card-based data display, CnCellRenderer for intelligent value formatting, CnObjectCard for individual card rendering, and CnKpiGrid for dashboard metrics.

## Scope
- CnCardGrid, CnObjectCard, CnCellRenderer, CnKpiGrid
- CSS modules and barrel export updates
